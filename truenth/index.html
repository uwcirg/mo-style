Tooltips and Popovers

Example code for tooltips:
<span class="show-tooltip" data-toggle="tooltip" data-placement="left" title="Tooltip title"> blabla </span>

NOTE:
  • The show-tooltip class is MANDATORY
  • The data-toggle="tooltip" attribute is MANDATORY
  • The title attribute is MANDATORY (this attribute will be displayed as the content of the tooltip)
  • The data-placement="left" attribute is optional (you can choose among "right", "left", "top", "bottom")

Popovers:

Example code for popovers:
<span class="show-popover" data-toggle="popover" data-placement="left" data-content="Popovers content"> blabla </span>

NOTE:
  • The show-popover class is MANDATORY
  • The data-toggle="popover" attribute is MANDATORY
  • The data-content attribute is MANDATORY (this attribute will be displayed as the content of the popover)
  • The data-placement="left" attribute is optional (you can choose among "right", "left", "top", "bottom")
  • The title attribute is optional (this attribute sets the title of the popover)

We need to use 3 Javascript hooks to make the overlay feature work

There are text overlay JS and SCSS files for this widget to work

The JS file can be loaded on the particular page using the requireScript helper function

  • js-text-overlay = to link the main tile
  • js-text-overlay--content = to link the hidden content that will be shown when the overlay is activated
  • js-text-overlay--title = to add the icons next to the tile and trigger the overlay effect

Note: If any additional colour styles need to be added, separate classes need to be added in the markup. The classes with -js prefix should not be modified.

Below is the barebones markup for the feature to work

Code to use:

    <div class="row js-text-overlay">
        <div class="col-xs-12 set-child-img-flex-center no-padding">
            <img src="http://lorempixel.com/g/200/200" class="img-responsive"/>
            <div class="js-text-overlay--content">
                <div class="col-xs-12">
                    <p>Rhoncus a neque a laoreet a torquent a adipiscing vestibulum volutpat.</p>
                </div>
            </div>
        </div>
        <div class="col-xs-12 no-padding">
            <div class="js-text-overlay--title">
                <h5>Vestibulum vehicula</h5>
                <span class="js-text-overlay--icon fa fa-info-circle"></span>
            </div>
        </div>
    </div>
